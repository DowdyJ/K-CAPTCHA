KB-CAPTCHA Demo

Welcome to the demo for the KB-CAPTCHA solution!

This webpage serves as a demonstration of the functionality of the CAPTCHA solution. Feel free to jump right into the test down below, or read about the motivation behind the project here.

The test below will record your keystrokes and make a prediction on your humanity! It does this by analyzing the following:

  • Average value and standard deviation for the duration that you hold down keys
  • Median value and interquartile range (IQR) for the time between keystrokes
  • How often your keys overlap when you type (two consecutive keys pressed at the same time)

This system is content agnostic! That means you can type anything into the box. However, spamming your keyboard with random input or using very short inputs are unlikely to produce typical typing behavior. For best results, try typing a sentence or two.
